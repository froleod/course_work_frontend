О сайте

Этот сайт - курсовая работа на тему "Интернет-ресурс о реляционных базах данных." Разработал Фролов Леонид - студент ИКБО-20-22. На сайте представлена главная страница с краткой информацией о самых популярных БД. Для каждой из них есть отдельные страницы с более подробной информацией. Была разработана межстраничная навигация, созданы CSS стили, использован JavaScript для создания интерактивной формы выбора БД.
